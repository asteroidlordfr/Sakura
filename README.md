# 🌸 Sakura
Sakura is an open-sourced Minecraft hack, a project made by the community - for the community.

We allow pull requests, so we'd be glad to look at your ideas!
We also review the ``Issues`` tab, find an issue or want to suggest something then that's your place!

If you contribute via a pull request, please follow these standards:
- Follow Java coding conventions
- Test your modules that you add
- Don't add anything that could harm Sakura's users

### Keybinds (Default)
| Action | Key |
|--------|-----|
| Open GUI | `RIGHT_SHIFT` |
| Toggle KillAura | `R` |
| Toggle Flight | `F` |
| Toggle ESP | `X` |
| Toggle Speed | `V` |

*You can change keybinds in Minecraft's controls menu under "Sakura"*

## 💻 Support

| Version | Loader | Supported |
|:----------|:------------------------|:---------------|
| `Java 1.21.11` | Fabric | `✅` |

As of now, we only support one version and mod loader as this is made by a solo dev.
*Minecraft: Bedrock Edition* is **NOT** supported at this time, however we plan on it for the future. 👀

## 🚀 Installation

1. **Install Fabric** for any supported Minecraft version from [FabricMC](https://fabricmc.net/)
2. **Download** the latest Sakura release from the [Releases](https://github.com/asteroidlordfr/Sakura/releases) tab
3. **Place** the `.jar` file in your `.minecraft/mods` folder
4. **Launch** Minecraft with the Fabric profile
5. **Press** `RIGHT_SHIFT` to open the GUI
